Python Snake

A classic Snake game built with Python and the turtle graphics module. This was an AI class project with difficulty selection, enemy snakes, and search-based AI behaviors.

About the game

You control the snake with W, A, S, D. Eat food to grow and avoid walls and other snakes. Choose your difficulty at the start. The game uses a grid-based board and includes simple reflex and search-based AI for enemy behavior (see the code for details).

Tech stack

  • Python 3
  • turtle (graphics and game window)
  • Search / AI logic for enemy movement (e.g. avoiding walls and collisions)

How to run locally

The game runs in a desktop window, not in the browser. From the project root: 

cd PythonSnake
python3 main.py

Use W (up), A (left), S (down), and D (right) to move. Select difficulty when the game starts.
